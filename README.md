🚀 Overview

The HR Attrition Analytics Dashboard is a data-driven project that analyzes employee attrition using Python and Excel to uncover insights and support HR decision-making.

This project helps organizations understand:

Why employees leave
Which departments are most affected
How to reduce attrition using data-driven strategies
